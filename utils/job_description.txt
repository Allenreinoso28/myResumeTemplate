Job Description:	
Overview


The Full-Stack Software Developer will support the design and development of internal software solutions used across our consulting operations. The student will work on modern full-stack web applications that streamline workflows, improve operational efficiency, and enhance collaboration within the organization. This role offers hands-on experience across the entire software development lifecycle (SDLC), including planning, development, testing, deployment, and maintenance of production-grade tools used by our team.

Responsibilities (including but not limited to)


Develop and maintain internal full-stack applications supporting consulting delivery and operational processes

Build front-end features using modern JavaScript frameworks, HTML, CSS, and responsive design practices

Develop scalable back-end services using Node.js, REST APIs, and MongoDB or comparable databases

Support integration of internal tools with third-party platforms and APIs to improve automation and data flow

Participate in code reviews, write clean and maintainable code, and follow Git-based version control best practices

Test, debug, and troubleshoot applications to ensure reliability and performance

Assist with documenting technical specifications, architecture decisions, system workflows, and user guides

Collaborate with stakeholders to gather requirements and translate business needs into technical solutions

Support deployment of applications in development and production environments, including basic Linux/Unix interaction when required


Evaluation Criteria


Enrolled in a post-secondary program (Computer Science, Software Engineering), Canadian Citizen

Experience with JavaScript, HTML, and CSS

Experience with React

Familiarity with Node.js, REST APIs, and database technologies such as MongoDB

Understanding of core software development principles, Git version control, and debugging techniques

Ability to learn new technologies quickly and work independently in agile or hybrid

Strong problem-solving skills and attention to detail
